# Project History

The full internal decision log is maintained privately by the operator. The
public history is intentionally concise:

1. OpenCairn started as a personal and team knowledge OS with workspaces,
   projects, pages, source ingest, graph search, and AI agents.
2. The foundation established a separate Hono API, Next.js web app, Temporal
   worker, Drizzle/Postgres database layer, LLM provider abstraction, and Yjs
   collaboration server.
3. The editor grew from solo notes into collaborative pages with comments,
   mentions, sharing, permissions, and richer blocks.
4. Ingest expanded from upload parsing into source notes, embeddings, import
   workflows, and evidence-aware retrieval.
5. The agent platform moved from stubbed chat surfaces toward runtime agents,
   scoped retrieval, document editing commands, and grounded answer checks.
6. The web product surface was reorganized around a durable app shell, canonical
   workspace/project/page URLs, and discoverable settings.
7. The public repository is now curated for OSS contributors and self-hosters;
   raw implementation plans and review logs are kept private.

See `docs/contributing/roadmap.md` for current status.
