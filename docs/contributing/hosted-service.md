# Hosted Service

OpenCairn is open-source (AGPLv3). Sungblab also operates a managed hosted service at **opencairn.com**.

## What's in this repo

- Application source code
- Database schema and migrations
- Configuration examples (`.env.example`)
- Landing page (`landing/`)
- All technical documentation under `docs/`

## What's not in this repo

The following are specific to the hosted service and live at opencairn.com:

| Content | URL |
|---------|-----|
| Privacy Policy | opencairn.com/legal/privacy |
| Terms of Service | opencairn.com/legal/terms |
| Pricing & Billing | opencairn.com/pricing |
| Blog | opencairn.com/blog |

## Self-hosting

If you're self-hosting OpenCairn, you are responsible for your own legal documents and billing setup. The application code does not include any hosted-service-specific content.

See `docs/contributing/dev-guide.md` for setup instructions.
